TVS News Portal DevelopmentToggle View

from ispyhumanfly

TVS originally started off as an attempt to create a different kind of news portal. Built upon ideas that derived from such sites as digg, slashdot, whitesoap and various blogs that I had come across. TVS was to be an easy to use and intuitive news portal. However with the nudging of friends and fellow developers (and while still maintaining the original intent of the site), TVS is now an actively developed news portal software being made freely available to anyone interested. And nearing alpha release under a yet to be determined open source license.

TVS is written in Object-Oriented Perl and is built upon the CGI::Builder framework. It uses XML for configuration files and the Perl DBI module for seemless integration with most ANSI compliant SQL servers. The template system is built upon the Perl HTML::Template module and the script.aculo.us Web 2.0 JavaScript framework for a fully customizable and modern style system.
